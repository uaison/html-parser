# html-parser
Simple HTML to JSON parser use Regexp and String.indexOf

## Basic usage

```javascript

var html = "<div style='height:10rpx;width: 20rpx;'>1<p>2<br/><a href='http://www.baidu.com'>3</a></p><p>2</p></div>"
htmlParser(html)

```
